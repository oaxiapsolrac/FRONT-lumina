Lumina-IA 🌟
Lumina-IA é uma aplicação interativa que utiliza modelos de linguagem de última geração (como a GROQ API) para oferecer uma experiência de chat inteligente e fluida. O projeto foca em fornecer uma interface amigável e respostas rápidas para auxiliar usuários em diversas tarefas, desde dúvidas gerais até auxílio com código.

🚀 O que o projeto faz?
O Lumina-IA funciona como um assistente pessoal virtual. Suas principais funcionalidades incluem:

Processamento de Linguagem Natural: Compreende e responde perguntas complexas em diversos idiomas.

Interface Intuitiva: Desenvolvido com tecnologias modernas de frontend para garantir uma navegação responsiva.

Integração com APIs de IA: Utiliza modelos poderosos para geração de texto e análise de dados.

Histórico de Conversas: (Se aplicável) Permite que o usuário visualize interações passadas.

🛠️ Tecnologias Utilizadas
Este projeto foi construído utilizando as seguintes tecnologias:

React.js ou Vite - Framework de interface.

Tailwind CSS - Estilização moderna e responsiva.

GROQ API - O "cérebro" por trás das respostas da IA.

Node.js - Ambiente de execução para o desenvolvimento.

📦 Como instalar e rodar o projeto
Siga os passos abaixo para configurar o ambiente localmente:

Clone o repositório:

Bash
git clone https://github.com/oaxiapsolrac/lumina-ia.git
Acesse a pasta do projeto:

Bash
cd lumina-ia
Instale as dependências:

Bash
npm install
# ou
yarn install
Configure as variáveis de ambiente:
Crie um arquivo .env na raiz do projeto e adicione sua chave da API:

Snippet de código
GROQ_GEMINI_API_KEY=SUA_CHAVE_AQUI
Inicie o servidor de desenvolvimento:

Bash
npm run dev
Abra o navegador em http://localhost:5173 (ou na porta indicada no terminal).
